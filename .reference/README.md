# .reference/

이 폴더는 **콘텐츠 마이그레이션용 원본 자료**입니다.

빌드 대상이 아니며, 코딩 에이전트가 프로젝트 MDX 콘텐츠를 작성할 때 참조합니다.

## 구조

```
.reference/
├── resume/
│   └── 고재현_이력서_2025_포트폴리오포함.pdf    # 기존 이력서 (2025.01, 시나몬 미반영)
└── notion-export/
    ├── 고재현_포트폴리오_*.html                  # 메인 포트폴리오 페이지
    ├── 숏폼_제작_AIaaS_*.html                   # 프로젝트별 상세 페이지 (18개)
    └── ...
```

## 용도

- **resume/**: 이력서 JSON 데이터 작성 시 원본 참조
- **notion-export/**: 프로젝트 MDX 콘텐츠 마이그레이션 시 원본 텍스트, 구조, 에셋 참조

## 주의사항

- 이 폴더는 `.gitignore`에 포함하지 **않음** (에이전트 접근 보장)
- Next.js 빌드에서 dot 폴더는 자동 제외됨
- NDA 관련: 기존 노션 포트폴리오에 이미 공개된 내용만 포함
