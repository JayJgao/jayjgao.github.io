sequenceDiagram
    participant U as User
    participant A as Agent
    participant H as Highlight API
    participant S as Subtitle API
    participant D as Design API
    participant E as Editing API

    U->>A: Provide Videos & Description
    loop Feedback Loop (until confirmed complete)
        A->>U: Ask: "Which Highlight Persona?"
        U->>A: Answer: "..."
        A->>U: Ask: "Subtitle Language?"
        U->>A: Answer: "..."
        A->>U: Ask: "Design Style?"
        U->>A: Answer: "..."
        A->>U: Ask: "Text Font & Position?"
        U->>A: Answer: "..."
        A->>U: Ask: "Video Ratio?"
        U->>A: Answer: "..."
        A->>U: Confirm: "Is the feedback complete?"
        U->>A: Confirmation: "Yes" (or "No" → loop again)
    end
    A->>H: Call Highlight API (with parameters)
    H-->>A: Highlight Results
    A->>S: Call Subtitle API (with parameters)
    S-->>A: Subtitle Results
    A->>D: Call Design API (with parameters)
    D-->>A: Design Results
    A->>E: Call Editing API (with parameters)
    E-->>A: Editing Results
    A->>U: Deliver Final Video
