## 🚧 Problem
텍스트 기반 질의에서 영상 후보를 빠르게 탐색하기 위한 retrieval 품질이 낮아,
데모 단계에서도 정확한 결과 도달까지 반복 질의가 많았습니다.

## 💡 Approach
Text-to-Video retrieval 파이프라인을 재설계해
질의 해석 -> 후보 생성 -> 재정렬 단계를 분리했습니다.

## ⚙️ Execution
- 질의 의도 분해(주제/행동/스타일) 기반 검색 키 확장
- 임베딩 검색 + 규칙 기반 rerank 하이브리드 적용
- 데모 시나리오 기준 평가셋을 구성해 반복 튜닝

## ✅ Results
- Meta Llama 해커톤 Text-to-Video Retrieval 부문 3위
- 실시간 데모 탐색 안정성 개선
